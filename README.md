# 🛡️ Nivesh Mitra: Vernacular AI-Powered FD Advisor

**Nivesh Mitra** is a smart financial companion designed for Bharat. It simplifies Fixed Deposit (FD) decisions for the next billion users by combining **Generative AI** with real-time financial data, all in a user-friendly **Hinglish** interface.

## ✨ Key Features
* **🗣️ Vernacular AI Advisor:** Ask questions in Hindi/Hinglish (e.g., *"SBI vs HDFC, kahan milega zyada byaaj?"*) and get instant, voice-enabled advice.
* **📈 Real-Time Comparison:** Live interest rate tracking across Public, Private, and NBFC sectors (Shriram Finance, Bandhan, SBI, etc.).
* **📉 Inflation Reality Check:** A unique visualizer that shows how your idle cash loses value over time compared to FD growth.
* **👥 Persona-Based Advice:** Tailored recommendations for Students, Professionals, and Senior Citizens with automated 0.50% bonus calculations.
* **💎 Premium UI:** A sleek, modern dashboard built for high-performance and ease of use.
